HTML dan CSS

Skill pertama saya adalah menguasai html dan css, dimana html berfungsi sebagai kerangka awal dari sebuah website, sementara css berfungsi sebagai media untuk memberikan efek dan styling untuk sebuah website. Dua skill atau keahlian ini merupakan dasar dari semuanya.

Javascript

Selanjutnya saya mempunyai skill dan basic logic dari javascript. Javascript adalah salah satu bahasa programming yang sangat populer dan sangat sering digunakan dalam development website. Skill ini sangat dibutuhkan di dunia kerja dan akan sangat berguna di industri yang mengedepankan web/app development.

Bootstrap

Skill lain saya adalah menguasai dan memahami penggunaan dari bootstrap. Bootstrap adalah sebuah library / framework css yang sangat mudah dan praktis. Bootstrap sendiri digunakan untuk memudahkan developer untuk melakukan styling dan adjusting sebuah website. Skill ini dibutuhkan karena dapat mempersingkat waktu untuk membuat sebuah project.

Tailwind CSS

Saya juga mampu untuk menerapkan dan menggunakan salah satu library / framework dari css, yaitu Tailwind CSS. Tailwind css mirip dengan bootstrap, perbedaan nya terletak pada element nya. Di bootstrap kita disajikan element atau property html lengkap dengan styling nya. Sedangkan di tailwind, semua element nya di styling secara manual menggunakan class yang ada.

Photography

Photography merupakan salah satu additional skill saya selain menguasai operating system lain. Di photography saya menguasai beberapa teknik foto dasar dan teknik color grading dasarnya. Dan ini adalah salah satu hasil dari foto yang saya tangkap dan saya coloring sendiri
